# Checkbox

A custom checkbox and radio button component with consistent styling and accessibility features.

## Features

- ✅ Checkbox and radio button variants
- ✅ Custom styled with smooth animations
- ✅ Label support (string or JSX)
- ✅ Error state handling
- ✅ Disabled state
- ✅ Dark mode support
- ✅ Full accessibility (WCAG compliant)

## Accessibility

- Proper label association with `htmlFor` and `id`
- Error messages with `role="alert"`
- `aria-invalid` for error states
- `aria-describedby` linking to error messages
- Full keyboard support (Space to toggle)
- Focus indicators
