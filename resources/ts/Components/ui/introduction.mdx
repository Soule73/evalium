# Examena UI Components

Welcome to the Examena UI component library documentation. This library provides a collection of reusable React components built with **Tailwind CSS v4** for the Examena online exam management platform.

## Getting Started

All components are built with:

- **React 19** with TypeScript
- **Tailwind CSS v4** for styling
- **Heroicons** for icons
- **Accessibility** best practices

## Component Categories

### Form Components

- **Select** - Searchable dropdown with keyboard navigation
- **Textarea** - Multi-line text input with validation
- **TextEntry** - Display label-value pairs

### Feedback Components

- **Badge** - Status indicators (success, error, warning, info, gray)
- **AlertEntry** - Alert messages with icons
- **Modal** - Dialog overlays with various sizes
- **ConfirmationModal** - Confirmation dialogs for critical actions

### Layout Components

- **Section** - Content sections with optional collapsible functionality
- **Button** - Primary action buttons with variants

## Usage

Import components from the `@examena/ui` package:

```tsx
import { Button, Badge, Modal, Select } from "@examena/ui";

function MyComponent() {
    return (
        <div>
            <Button color="primary">Click me</Button>
            <Badge label="Active" type="success" />
        </div>
    );
}
```

## Design Principles

1. **Consistency** - All components follow the same design language
2. **Accessibility** - WCAG 2.1 compliant
3. **Flexibility** - Customizable through props and className
4. **Performance** - Optimized for production use
5. **Type Safety** - Full TypeScript support

## Color Palette

The components use a consistent color system:
