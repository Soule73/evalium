# Input

A flexible text input component with label, error handling, and helper text support. Includes full accessibility features and dark mode support.

## Features

- ✅ Multiple input types (text, email, password, number, tel, url, search)
- ✅ Label with proper association
- ✅ Error states with ARIA invalid
- ✅ Helper text for additional context
- ✅ Disabled state
- ✅ Required field support
- ✅ Dark mode support
- ✅ Full accessibility (WCAG compliant)

## Accessibility

- Proper label association with `htmlFor` and `id`
- Error messages announced with `role="alert"`
- `aria-invalid` attribute for error states
- `aria-describedby` linking to helper text and errors
- Full keyboard navigation support
