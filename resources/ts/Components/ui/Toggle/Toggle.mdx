# Toggle

A switch/toggle component for on/off states with multiple sizes, colors, and accessibility features.

## Features

- ✅ Multiple sizes (sm, md, lg)
- ✅ Multiple colors (blue, green, purple, red, gray)
- ✅ Optional label
- ✅ Status label display
- ✅ Disabled state
- ✅ Dark mode support
- ✅ Full accessibility with ARIA switch role
- ✅ Smooth animations

## Accessibility

- Uses proper `role="switch"` for screen readers
- `aria-checked` attribute reflects current state
- `aria-label` for screen reader context
- Label association with `htmlFor` and `id`
- Keyboard support (Space/Enter to toggle)
- Focus indicators
- Title attribute for hover context
